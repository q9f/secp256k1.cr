    Overview

    Provides an ECDSA Signature containing the random point #r, the signature proof #s, and the recovery id or #v value.

    Properties:

    • #r (Num): the x coordinate of a random point R.
    • #s (Num): the signature proof of a message.
    • #v (Num): the recovery id or #v value.
