    class Secp256k1::Key

    • Secp256k1::Key
    • Reference
    • Object

    Overview

    Provides a Secp256k1 keypair containing a secret number (private key) and a public point on the elliptic curve (public key).

    Properties:

    • #private_key (Num): the secret number representing the private key.
    • #public_key (Point): the point on the elliptic curve representing the public key.

    Defined in:

    secp256k1/key.cr
