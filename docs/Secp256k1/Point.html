    class Secp256k1::Point

    • Secp256k1::Point
    • Reference
    • Object

    Overview

    Provides a point in the two-dimensional space of any elliptic curve. In most cases, such a point on a given curve represents a public key. However, for keypairs, a Key type shall be used!

    Properties:

    • #x (Num): the position on the x-axis.
    • #y (Num): the position on the y-axis.
